  我的制作过程说明是：

  * 展示西工大的几个标志性建筑，并通过剪映实现场景与场景之间的转场。
  * 我通过PS，将几个建筑分离出来，放在开头展示。
  * 视频结尾意义不明的一卡通是想证明视频是我本人拍摄。

  我的心得体会是：

  - 剪映是一个很厉害的软件，可以做很多事情，但是也很复杂，需要多学习和练习。
  - 视频的制作过程需要有计划和创意，要根据内容和目标来选择素材、过渡和音频，并且要注意视频的逻辑和美感。
